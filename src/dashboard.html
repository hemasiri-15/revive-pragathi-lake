System Architecture

IoT → LoRa → Cloud → AI → Dashboard

Hardware Layer
☀
Solar Panel (5W)
Charges 3000 mAh LiPo. ~25 Wh/day at Hyderabad peak sunlight. Self-powered, floats anywhere on the lake.
🧠
Arduino Nano
Wakes every 60s, reads all 5 sensors, packages JSON payload, transmits via LoRa, sleeps. Sleep current ~20µA. Cost ₹350.
📡
LoRa SX1278 (5 km range)
433 MHz, 50 mW, no SIM card. Single gateway on the lake bund receives all 3 nodes. Module cost ₹450.
💧
5 Sensor Probes
pH, DO, turbidity (NTU), TDS, temperature. IP67 sealed. pH <6.5 = sewage. DO <5 = fish risk. DO <2 = dead zone.
Software + Data Layer
🖧
Gateway — Raspberry Pi 4
Receives LoRa packets, forwards to cloud via 4G + MQTT. Local SD fallback if cloud unreachable.
⚙
Backend — Node.js + TimescaleDB
Validates and stores readings. Runs threshold checks. Fires SMS/push alerts to GHMC officials via MQTT.
🤖
AI — Attention-LSTM + FastAPI
72-hour WQI prediction, bloom probability, anomaly detection. Monte Carlo dropout for uncertainty quantification.
🛰
Remote Sensing — GEE + Sentinel-2
Google Earth Engine pipeline. NDWI, NDVI, chlorophyll-a band ratios. Daily automated processing at 10m resolution.
